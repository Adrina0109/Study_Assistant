# Study Assistant AI

AI-powered tool to summarize notes, extract key points, and generate quizzes instantly.

Study Assistant AI is a full-stack web application designed to help students learn smarter and faster.  
Paste your study notes and the AI will:

- Generate a concise summary  
- Extract key learning points  
- Create Fill-in-the-Blank questions  
- Generate MCQs with options, answers, and explanations  

Built with Next.js, FastAPI, and Python NLP (spaCy + NLTK).

---

## Features

- AI-generated summary of long content
- Key points extraction to highlight important concepts
- Fill-in-the-blank quiz for recall-based learning
- MCQ quiz with options, correct answer, and explanation
- Works locally without external AI APIs

---

## Tech Stack

| Layer | Technology |
|--------|---------------|
| Frontend | Next.js, TypeScript, Tailwind CSS |
| Backend | FastAPI (Python) |
| NLP & AI | spaCy, NLTK |
